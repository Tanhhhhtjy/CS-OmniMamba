## 1. 项目概述

**任务：Seq→1 降水临近预报**

```
输入：RADAR[B, 10, 1, 66, 70]（t-9 … t）+ PWV[B, 1, 1, 66, 70]（t）
输出：RAIN[B, 1,  1, 66, 70]（t+1，即 6 分钟后）← dataset 字段 batch shape
      模型 forward 实际输出 [B, 1, H, W]（train.py 中对 rain 字段 .squeeze(1) 后送入模型/loss）
```

- 三模态统一归一化：`(255 - pixel) / 255.0`（255=无信号→0, 0=最强→1）
- 时间步长：6 分钟（STEP_SECONDS=360）
- 图像尺寸：H=66, W=70（array shape；PIL `Image.size` 返回 `(70,66)`，方向相反）

### 3.4 `src/dataset.py` — 数据集

**时间戳收集：**
```python
_collect_timestamps([radar_dir, pwv_dir, rain_dir])
# 取三目录 PNG 文件名（格式 "2025-05-01-00-06-00.png"）的时间戳交集
# 文件名→datetime 用 strptime("%Y-%m-%d-%H-%M-%S")
```

**样本索引：**
```python
build_sample_index(timestamps, T=10) -> List[int]
# range(n - T)：所有满足连续 T+1 步（各步差 = STEP_SECONDS 秒）的起始位置
# 用 .total_seconds() 而非 .seconds（避免跨天 bug）
```

**数据集类：**
```python
class RainDataset(Dataset):
    # __getitem__(idx) 返回：
    {
        "radar":        Tensor[T, 1, H, W],   # RADAR 历史 t-T+1…t
        "pwv":          Tensor[1, 1, H, W],   # PWV at t
        "rain":         Tensor[1, 1, H, W],   # RAIN at t+1（目标）
        "rain_current": Tensor[1, 1, H, W],   # RAIN at t（Persistence 基线用）
    }
```

**split 切分（重要）：**

切分按**目标帧** `timestamps[i + T].date()` 而非窗口起点 `timestamps[i].date()`，避免标签泄漏。实现在 `src/train.py` 的 `_filter_dataset_by_split()`。

### 3.5 `src/loss.py` — 加权 MSE

```python
weighted_mse_loss(pred, target) -> Tensor
# rain_mask = (target > RAIN_EPS).float()
```

雨区贡献约 49% loss（基于实际像素分布）。

### 3.6 `src/metrics.py` — 评估指标

```python
compute_csi_pod_far(pred, target, threshold) -> {"csi", "pod", "far"}
# NaN guard: CSI/POD 当 denom_pod==0（目标无雨）返回 NaN
#            FAR 当 denom_far==0（模型不预测任何雨）返回 NaN

compute_mse_mae(pred, target) -> {"mse", "mae_rain"}
# mae_rain 当无雨像素时返回 NaN

class MetricsAccumulator:
# 聚合原始 TP/FP/FN（不是 per-batch 平均），最后一次性计算
# .update(pred, target)  — 每 batch 调用
# .compute()             — 返回 csi_weak/pod_weak/far_weak/csi_strong/pod_strong/far_strong/mse/mae_rain
# .reset()
```

阈值定义：
- `THRESH_WEAK   = 2/255`（原像素 < 253）
- `THRESH_STRONG = 55/255`（原像素 < 200）

## 8. 关键设计决策备忘

| 决策 | 选择 | 理由 |
|------|------|------|
| 输入历史 | RADAR×10 + PWV×1（单帧） | PWV 相邻帧高度相关（MAE~0.34-0.63），单帧信息足够 |
| 归一化方向 | 反转（255→0, 0→1） | 与 PWV 物理量正相关（r=-0.997）一致 |
| RAIN 历史是否输入 | 否 | 任务定义简化（单步预测，不存在误差累积） |
| 损失函数 | 加权 MSE（雨区×10） | 雨区贡献约 49% loss，防退化为全无雨预测 |
| 评估主指标 | CSI（两阈值：weak/strong） | 对稀疏事件最合适 |
| Persistence | 强基线，非上界 | 降水持续性强（7月中位持续约 294 分钟） |
| SSIM/FSS | Phase 2 延期 | stub 阶段无意义，待有可比较模型后实现 |
| MetricsAccumulator | 聚合 TP/FP/FN | per-batch 平均在无雨 batch 存在时结果偏差大 |
| split 切分基准 | 目标帧时间戳 | 按窗口起点切分会导致标签泄漏 |
